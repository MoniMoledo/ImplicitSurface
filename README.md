# Computer Graphics Project

Using the base code given in class, create a graphic application to visualize implicit surfaces using "Marching Tetrahedra" algorithm.

It should:

1) Implement Phong shading on fragment shader.
2) Allow users to rotate the object around axel Y using 'L' and 'R' keys.
3) Allow users to rotate the object around axel X using 'C' and 'B' keys.
4) Allow users to change perspective projection to orthographic projection using 'O' key.

Java, OpenGL 3.2.0
